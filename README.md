# zpl-designer

브라우저에서 라벨을 시각적으로(WYSIWYG) 디자인하고 그 결과를 **ZPL II 코드**로
추출하는 라벨 디자이너 툴.

페이지 크기, 텍스트, 선, 박스, 원/타원, 대각선, 테이블, 이미지, 1D 바코드,
2D 코드(QR 등) 등 ZPL로 변환 가능한 모든 도형·레이아웃을 손쉽게 배치하고
즉시 ZPL 코드를 생성한다.

## 문서

- [제품 요구사항 정의서 (PRD)](docs/PRD.md) — 전체 기능·아키텍처·로드맵

## 기술 스택 (계획)

- React + TypeScript + Vite
- Konva.js (캔버스)
- IndexedDB (로컬 저장, 백엔드 없음)
